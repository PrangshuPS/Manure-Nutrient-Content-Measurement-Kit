# Low-Cost Manure Nutrient Content Measurement Kit

## Overview
A portable, low-cost system to analyze key nutrient parameters in organic manure directly on-site.

Designed to measure:
- Nitrogen (N), Phosphorus (P), Potassium (K)
- pH
- Moisture
- Color-based indicators

Built using ESP32 and integrated sensors, the system provides quick insights without the need for expensive lab testing.

---

## Features
- Real-time sensor data collection  
- On-device display using OLED  
- WiFi-based data upload (ThingSpeak)  
- Portable and cost-effective design  

---

## Tech Stack
- ESP32  
- pH Sensor  
- Capacitive Soil Moisture Sensor  
- TCS34725 RGB Sensor  
- OLED Display (SSD1306)  
- Arduino (C/C++)  

---

## Challenges
- Sensor calibration and accuracy
- Noise and environmental variations  
- WiFi connectivity stability  
- Color differentiation limitations
